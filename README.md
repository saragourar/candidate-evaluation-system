# Candidate Evaluation System – Assistant Professor B

## Project Description
This project is a web-based candidate evaluation system designed for Assistant Professor B recruitment.  
The system calculates candidate scores based on official academic evaluation criteria and stores the results in a database.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP Server

## System Features
- Candidate information input
- Scientific works evaluation
- Conference evaluation
- Professional experience evaluation
- Interview score calculation
- Automatic score calculation
- Data storage in database

## Project Structure
- index.html → User interface
- style.css → Design and layout
- script.js → Score calculation logic
- save.php → Data storage in database
- evaluation_db.sql → Database structure

## How to Run the Project

1. Install XAMPP
2. Copy project folder into:
   C:\xampp\htdocs\
3. Start Apache and MySQL
4. Open phpMyAdmin
5. Import evaluation_db.sql
6. Open browser
7. Run:
   http://localhost/evaluation-assistant-b

## Evaluation Criteria

Scientific works: 8 points  
Conferences: 3 points  
Professional experience: 3 points  
Interview: 4 points  

Total: 20 points
